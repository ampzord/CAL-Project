§ loadStreetInformation()

void SupermarketNetwork::loadStreetInformation ( )

Faz load das Ruas em txt atualizando o atributo TwoWays de cada uma.

Para cada Rua em txt,"True" ou "False" definem o atributo de cada objeto.

True torna possivel a Rua ter 2 sentidos.

False garante que a Rua so tem um sentido.
